# ASL Speed Run Game

A real-time American Sign Language (ASL) speed run game using MediaPipe for hand tracking.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open the Game**:
    - The terminal will show a local URL (e.g., `http://localhost:5173`).
    - Open that URL in your browser.

## How to Play

1.  **Start**: Show a **Thumbs Up** gesture to the camera.
2.  **Play**: Mimic the ASL letters displayed on screen as quickly as possible.
    - **A**: Fist
    - **B**: Open Palm
    - **L**: L Shape
    - **V**: Peace Sign
    - **Y**: Phone Gesture
